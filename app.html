complete_order(order_id)
Handles HTTP GET and POST requests for the `/complete_order/<order_id>` route.
 
Fetches an order from the database using `order_id`, deletes the order and its
corresponding items from the `Orders` and `Items` tables. If the order is cooked,
sets the `delivered` flag of the order to `True`, and commits the changes to the
database. If the order is not cooked, it only deletes the order and its corresponding
items from the database.It then redirects to the `waiter()` function which
renders the `waiter.html` template.
 
Parameters:
    order_id (int): The ID of the order to be completed.
